## Instructions

### Setup

```shell
git clone https://github.com/codelikeagirl29/react-portfolio.git
cd react-portfolio
```
and then to install and view in browser.

```shell
npm install
npm run start
```

Install and run -

```shell
yarn
yarn start
```

### How to use

- Open `src/info/` and make changes.
- Build and deploy -

```shell
yarn build
yarn deploy
```
